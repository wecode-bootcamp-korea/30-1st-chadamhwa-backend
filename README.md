## 프로젝트 소개

- 차담화는 이커머스 사이트인 술담화를 모티브로 구현한 웹 사이트 서비스입니다.
- 짧은 프로젝트 기간동안 개발에 집중하기 위해 기획 및 기능의 일정 부분을 비슷하게 구현했습니다.
